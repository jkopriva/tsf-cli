= {TSFName}

{TSFName} ({TSFShortName}) is a deployable instance of link:https://konflux-ci.dev[{KonfluxName}] that brings a secure software supply chain to your {OCPName} cluster. {TSFShortName} integrates {KonfluxName} with {RHTASName} and {RHTPAName} to provide end-to-end build, sign, and verify workflows for container images.

== What {TSFShortName} provides

* **Secure build pipelines** — Tekton-based CI/CD pipelines that produce signed container images with SLSA Level 3 provenance.
* **Artifact signing** — Automatic cryptographic signing of container images using {RHTASName} (Fulcio, Rekor, TUF).
* **Software bill of materials** — SBOM generation and vulnerability analysis through {RHTPAName}.
* **Identity management** — Red Hat build of Keycloak for single sign-on and access control.
* **Container registry integration** — Push built images to your Quay organization with automatic repository creation.
